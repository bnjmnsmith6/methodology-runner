## What was requested

- **Problem**: Users need a simple, interactive coin flip simulator accessible from any modern browser
- **Desired outcome**: Single HTML file that simulates coin flips with visual animation and result tracking
- **Success checks**: Click interaction triggers flip animation, displays heads/tails result, maintains persistent tally counter

## What was built

Created index.html — a single self-contained HTML file implementing a coin flip simulator. Clicking the coin triggers a CSS rotateY flip animation (900ms), randomly lands on heads or tails, displays the result, and maintains a running tally (heads / tails / total). A reset button clears the counters. No external dependencies.
