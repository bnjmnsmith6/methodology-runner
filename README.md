## What was built

Created index.html — a single-file HTML color palette generator with embedded CSS and JS. Displays 5 distinct random colors with hex labels on page load; clicking Generate produces a new set of 5 distinct colors. Color distinctness is enforced via RGB Euclidean distance (threshold 60) with a 200-attempt guard and fallback.
