## 🐛 Troubleshooting

### Issue: Worker won't restart
```bash
pm2 delete guppi-worker
pm2 start npm --name guppi-worker -- run worker
```

### Issue: Git conflicts on pull
```bash
git stash
git pull origin main
git stash pop
```

### Issue: README not appearing
- Check logs for "Failed to generate README" warnings
- Verify fs permissions in worktree directory
- Check that constellation packet exists at `.methodology/constellation-packet.md`

### Issue: Branch name not slugified
- Verify RP title is being passed to `ensureWorktree()`
- Check that `slugify()` function exists in `worktree-manager.ts`
- Look for log message "Using branch name: guppi-builds/..."
